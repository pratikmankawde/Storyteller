# libs

Place the ONNX Runtime GenAI Android AAR here for Qwen3-1.7B-Q4-ONNX support.

- **File:** `onnxruntime-genai-android-0.11.4.aar`
- **Download:** https://github.com/microsoft/onnxruntime-genai/releases/tag/v0.11.4  
  Direct link: https://github.com/microsoft/onnxruntime-genai/releases/download/v0.11.4/onnxruntime-genai-android-0.11.4.aar

If the AAR is missing, the app will not build (ONNX GenAI is required for the LLM). Place the AAR in `app/libs/`.
